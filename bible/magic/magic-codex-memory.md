# Codex / Memory Journals / Auratic Codices

## Identity

`Auratic codex` is the accepted technical term for the memory-journal magic category.

An auratic codex stores immersive experiential memories. It is accessed through a key incantation and aura/tether interaction, usually tied to the creator's aura or intent.

This file owns general codex mechanics. Alara's specific journal remains a separate object record under `object-alara-journal`.

## Core Mechanics

Codex memories are not ordinary written text or conventional encryption. A user focuses on the codex, recites the key incantation, and lets their aura press into the tether or access structure carved into the codex.

The resulting memory can feel like living the moment from within. It can carry sensory, emotional, bodily, and magical perspective, and can be vivid enough to overwhelm or endanger the user.

Codices do not store aura imprints by default.

## Elric's Tampering

Elric tampered with Alara's codex to make it anchor an aura-like imprint and function beyond normal codex behavior.

The corrupted key / bridge is canon. It lets an Elric-like presence interact through some memories, especially where Elric is represented or where he has enough connection.

The exact nature of that interactive presence remains unresolved. It may be an aura imprint, an Ink-linked remnant, a trap that imitates sentience, some partial Elric persistence, or a combination.

The bridge was not a normal codex feature.

## Codex Trap

Elric's codex trap intentionally idealizes or distorts Maya's parents. It reinforces Maya's belief that she can fix impossible losses and helps pull her toward fate/Skein knowledge.

This distortion should preserve the emotional truth that the memories contain real material, while recognizing that the presentation is manipulated.
