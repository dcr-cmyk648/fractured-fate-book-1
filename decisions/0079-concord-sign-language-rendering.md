# Concord Sign Language Rendering Convention

Date: 2026-06-30

## Status

Accepted future prose convention.

## Decision

Concord / Masked Clade sign language should be italicized in future manuscript prose.

Its translated style should not read exactly like clean spoken English. It should have a slightly ASL-like, compressed, awkward quality.

The underlying system is that certain common words have established signs, while the rest is phonetically rendered through zän/sign components. This can produce phrasing that feels partly direct and partly component-based.

Do not edit manuscript prose now. Apply this during later authorized prose revision.

## Rationale

This preserves sign language as a distinct communication system with ties to zän and Masked Clade culture, while giving later prose work a readable convention for differentiating signed dialogue from ordinary speech.
