# 0024: Auratic Codex Memory Mechanics

Date: 2026-06-24

Status: Accepted

## Decision

`Auratic codex` is the accepted technical term for memory-journal magic.

Codices store immersive experiential memories accessed by key incantation plus aura/tether interaction.

Codices do not store aura imprints by default. Elric tampered with Alara's codex to make it anchor an aura-like imprint and function as a corrupted bridge.

Elric's corrupted key/bridge is canon. The exact nature of the interactive Elric presence remains unresolved.

The codex trap intentionally idealizes or distorts Maya's parents, reinforcing her belief that she can fix impossible losses.

Niall showing Maya his codex to explain Alara's death remains candidate.

Harvick having an Elric-linked codex remains candidate.

`object-alara-journal` remains separate from this magic record; `magic-codex-memory` owns general codex mechanics only.

## Consequences

- Later Codex references should not imply all codices naturally anchor aura imprints.
- Elric's bridge should be treated as tampering with Alara's codex, not a normal codex function.
- Later Maya/Niall/Alara reviews should account for parent-memory idealization or distortion.
- Later Elric/Ink/Skein reviews should resolve the nature of the interactive Elric presence.
