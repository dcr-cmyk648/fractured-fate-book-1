# Candidate Material: Registry

## Technical Mechanics

Accepted high-level facts:

- passive mode identifies and alarms
- active mode identifies and triggers lethal response
- active mode has limited short-period discharge capacity
- the Heart provides most current Registry power
- the original Registry was not Heart-designed and likely had passive, limited-scope assumptions

Preserve for later:

- exact active/passive mode boundaries
- alarm types and thresholds
- lethal response mechanism
- how location pulse differs from intruder alarm
- how much active discharge can be stored
- how Caius survives or interacts with active-mode attacks
- how gateway-only detection would work

## Heart and Power Path

Preserve for `object-seraph-heart` and `magic-zan-lattices`:

## Bypass and Ink

Accepted high-level facts:

- Davian adds Arrek's name and relies on the Ink Pact/Boon

Preserve for `magic-ink-boons`:

- how long the bypass lasts
- why the Registry sees the combination as valid
- what signs Aldira/Alden could have noticed earlier
